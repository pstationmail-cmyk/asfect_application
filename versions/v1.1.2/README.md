# プレイリスト作成アプリ v1.1.2

## ダウンロード

- 推奨: `プレイリスト作成アプリ_v1.1.2.dmg`
- 予備: `プレイリスト作成アプリ_v1.1.2.zip`

## インストール

DMGを開き、`プレイリスト作成アプリ.app` を `Applications` フォルダへドラッグしてください。

## 主な変更

- v1.1.1のDMG内アプリに古い実行ファイルが混入していた問題を修正
- クリーンなアプリ束からDMG/ZIPを作り直し
- DMG内アプリの署名検証が通ることを確認
- アプリアイコンを新アイコンに差し替え
