# プレイリスト作成アプリ v1.1.1

> 緊急差し替え済み: v1.1.1の初回配布DMGに古い実行ファイルが混入していたため、このフォルダ内のDMG/ZIPはv1.1.2相当の正常ビルドに差し替えています。

## ダウンロード

- 推奨: `プレイリスト作成アプリ_v1.1.1.dmg`

## インストール

DMGを開き、`プレイリスト作成アプリ.app` を `Applications` フォルダへドラッグしてください。

## 主な変更

- チェック項目の文字色を明示し、macOSの外観差で消えにくい表示に修正
- 作成するフォルダ名の入力欄の文字色と背景を明示
- オプション欄を安定した2列レイアウトに変更
- ヘッダー右側の飾り文字を削除
- 作成するフォルダ名の初期値を空欄に変更
- v1.1.1 URLを開いた場合も、v1.1.2相当の正常ビルドを取得できるように差し替え
- アプリアイコンを新アイコンに差し替え
