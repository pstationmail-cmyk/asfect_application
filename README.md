# asfect_application

ASFECT向けの配布アプリ置き場です。

## 最新版

- アプリ名: プレイリスト作成アプリ
- バージョン: v1.0
- 配布ファイル: `versions/v1.0/プレイリスト作成アプリ_v1.0.dmg`

## インストール方法

1. `versions/v1.0/プレイリスト作成アプリ_v1.0.dmg` をダウンロードします。
2. DMGを開きます。
3. `プレイリスト作成アプリ.app` を `Applications` フォルダへドラッグします。
4. `Applications` フォルダからアプリを起動します。

アップデート時も同じ手順です。
新しいDMGを開いて `Applications` フォルダへドラッグし、確認が出たら既存アプリを置き換えてください。

## フォルダ構成

```text

versions/
```

## 注意

このアプリはローカルで動作します。PDF、素材フォルダ、コピー先フォルダは利用者が毎回選択します。
