# 修正ログ

## v1.1

- Excel `.xlsx` の読み込みに対応
- Excelでは「再生素材」列を優先し、同じ行の「演目」をタイトルとして利用
- `[V202]` / `【V202】` / `V202` 形式の動画番号抽出に対応
- QLab 5ワークスペース `.qlab5` の自動作成に対応
- GitHub上の `update.json` を使ったアップデート確認表示に対応
- アップデートがある場合、画面上部に「アップデートがあります。」と更新ボタンを表示

## v1.0

- PDFから動画番号とタイトルを抽出
- 動画番号は「アルファベット1文字 + 数字」に対応
- 素材フォルダから該当動画を検索
- PDF順に動画をコピー
- ファイル名への連番付与に対応
- m3uプレイリスト作成に対応
- コピー進捗、経過時間、残り目安、コピー中ファイル名を表示
- PDF上段タイトルを作成フォルダ名の初期値に反映
- 動画が非検出の場合も、確認後に実行できる分岐を追加
- DMG形式とZIP形式の配布ファイルを作成
